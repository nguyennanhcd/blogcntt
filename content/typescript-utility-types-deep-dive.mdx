## **TypeScript Utility Types: Your Type-Level Superpowers**

You don’t need to write every type from scratch.

TypeScript ships with **built-in utility types** — like `Partial<T>`, `Pick<T, K>`, `Record<K, T>`, and even **union manipulators** like `Exclude<T, U>`.

They **transform**, **filter**, and **compose** types with surgical precision.

Let’s go **deep**. With **real-world examples** you’ll use every day.

### **`Partial<T>`: Update Without Full Object**

<CodeBlock
  language="ts"
  highlight="5"
  collapsible
  defaultCollapsed={false}
>
{`interface User {
    id: string;
    name: string;
    email: string;
    role: 'admin' | 'user';
}

function updateUser(id: string, changes: Partial<User>) {
  // Only update provided fields
}`}
</CodeBlock>


### **`Required<T>`: Enforce All Fields**

<CodeBlock
  language="ts"
  highlight="2"
  collapsible
  defaultCollapsed={false}
>
{`interface Config {
    apiUrl?: string;
    timeout?: number;
}

const appConfig: Required<Config> = {
    apiUrl: 'https://api.example.com',
    timeout: 5000,
};`}
</CodeBlock>


### **`Pick<T, K>`: Extract What You Need**

<CodeBlock
  language="ts"
  highlight="3"
  collapsible
  defaultCollapsed={false}
>
{`interface User {
    id: string;
    name: string;
    email: string;
    password: string;
}

type PublicUser = Pick<User, 'id' | 'name' | 'email'>;`}
</CodeBlock>


### **`Omit<T, K>`: Remove What You Don’t Want**

<CodeBlock
  language="ts"
  highlight="2"
  collapsible
  defaultCollapsed={false}
>
{`type UserWithoutPassword = Omit<User, 'password'>;
// → { id: string; name: string; email: string; }`}
</CodeBlock>

### **`Exclude<T, U>`: Filter Out Types**

<CodeBlock
  language="ts"
  highlight="3"
  collapsible
  defaultCollapsed={false}
>
{`type Event = 'click' | 'hover' | 'scroll' | 'keydown';

type MouseEvent = Exclude<Event, 'keydown'>;
// → 'click' | 'hover' | 'scroll'`}
</CodeBlock>


### **`Extract<T, U>`: Keep Matching Types**

<CodeBlock
  language="ts"
  highlight="2"
  collapsible
  defaultCollapsed={false}
>
{`type Primitive = string | number | boolean | null;

type Truthy = Extract<Primitive, boolean | number | string>;
// → string | number | boolean`}
</CodeBlock>


### **`NonNullable<T>`: Remove null/undefined**

<CodeBlock
  language="ts"
  highlight="2"
  collapsible
  defaultCollapsed={false}
>
{`type ApiResponse = User | null;

function processUser(user: NonNullable<ApiResponse>) {
  // user is guaranteed to be User
}`}
</CodeBlock>


### **`ReturnType<T>`: Get What a Function Returns**

<CodeBlock
  language="ts"
  highlight="4"
  collapsible
  defaultCollapsed={false}
>
{`function getUser() {
    return { id: '1', name: 'Alex' };
}

type UserResult = ReturnType<typeof getUser>;
// → { id: string; name: string; }`}
</CodeBlock>


### **`Parameters<T>`: Get Function Args**

<CodeBlock
  language="ts"
  highlight="3"
  collapsible
  defaultCollapsed={false}
>
{`function apiCall(url: string, method: 'GET' | 'POST') { }

type ApiParams = Parameters<typeof apiCall>;
// → [url: string, method: 'GET' | 'POST']`}
</CodeBlock>


### **`InstanceType<T>`: Class Instance Type**

<CodeBlock
  language="ts"
  highlight="3"
  collapsible
  defaultCollapsed={false}
>
{`class UserService {
    getUser() { return { name: 'Alex' }; }
}

type Service = InstanceType<typeof UserService>;
// → UserService`}
</CodeBlock>

## **Don’t Do This**

<CodeBlock
  language="ts"
  highlight="1-4"
  collapsible
  defaultCollapsed={false}
>
{`// 1. Manual duplication
interface UserDTO { id: string; name: string; email: string; }

// 2. any for maps
const cache: any = {};

// 3. Wrong Exclude
type Bad = Exclude<string | number, string | boolean>;`}
</CodeBlock>


## **Do This Instead**

<CodeBlock
  language="ts"
  highlight="2, 5"
  collapsible
  defaultCollapsed={false}
>
{`// 1. Reuse with Pick/Omit
type UserDTO = Pick<User, 'id' | 'name' | 'email'>;

// 2. Record for lookups
const routes: Record<string, () => void> = {};

// 3. Combine for complex types
type UpdatePayload = { id: string } & Partial<User>;`}
</CodeBlock>


## **Utility Types Cheat Sheet**

* `Partial<T>` → **optional props**  
* `Required<T>` → **all required**  
* `Pick<T, K>` → **select keys**  
* `Omit<T, K>` → **remove keys**  
* `Record<K, T>` → **typed map**  
* `Exclude<T, U>` → **remove from union**  
* `Extract<T, U>` → **keep from union**  
* `NonNullable<T>` → **remove null/undefined**  
* `ReturnType<T>` → **get function return**  
* `Parameters<T>` → **get function args**  
* `InstanceType<T>` → **get class instance type**  
* `ConstructorParameters<T>` → **get constructor args**
