## **CSS Grid: The Modern Layout Revolution**

CSS Grid has **revolutionized** how we build web layouts.  
Forget floats, complex flexbox hacks, or fragile table layouts — **Grid gives you full 2D control** over your design.

Once you master its core concepts, you'll wonder how you ever lived without it.


## **The Foundation: Container & Items**

Every grid starts with a **container** and its **items**.

Set `display: grid` on the parent → all **direct children** become **grid items**.

<CodeBlock
  language="css"
  highlight="3-5"
  collapsible
  defaultCollapsed={false}
>
{`.container {
    display: grid;
    grid-template-columns: 1fr 1fr;
    gap: 1rem;
}`}
</CodeBlock>

> **Pro tip**: Only **direct children** are grid items. Nested elements? Use a new grid inside.


## **The Magic of `fr` Unit**

The `fr` unit = **fraction of available space**.

<CodeBlock
  language="css"
  highlight="2"
  collapsible
  defaultCollapsed={false}
>
{`grid-template-columns: 1fr 2fr;`}
</CodeBlock>
→ First column: 1 part  
→ Second column: 2 parts → **twice as wide**

<CodeBlock
  language="css"
  highlight="2"
  collapsible
  defaultCollapsed={false}
>
{`grid-template-columns: 1fr 1fr 1fr;  /* 3 equal columns */
grid-template-columns: 2fr 1fr;      /* 2:1 ratio */`}
</CodeBlock>


## **Grid Lines: Your Invisible Ruler**

Grid creates **numbered lines** around cells.

Use `grid-column` and `grid-row` to place items **anywhere**.

<CodeBlock
  language="css"
  highlight="3-4"
  collapsible
  defaultCollapsed={false}
>
{`.hero {
    grid-column: 1 / 3;   /* span from line 1 to line 3 */
    grid-row: 1 / 2;
}`}
</CodeBlock>

Use `span 2` to span across 2 columns/rows:

 <CodeBlock
  language="css"
  highlight="3-7"
  collapsible
  defaultCollapsed={false}
>
{` grid-column: span 2;`}
</CodeBlock>


## **Named Grid Areas: Layout as Art**

Define your layout **visually** with `grid-template-areas`.

<CodeBlock
  language="css"
  highlight="3-7"
  collapsible
  defaultCollapsed={false}
>
{`.container {
    display: grid;
    grid-template-areas:
        "header header"
        "sidebar main"
        "footer footer";
    grid-template-columns: 250px 1fr;
    grid-template-rows: auto 1fr auto;
}`}
</CodeBlock>

Then assign items:

<CodeBlock
  language="html"
  highlight="2-4"
  collapsible
  defaultCollapsed={false}
>
{`<header style="grid-area: header">...</header>
<aside style="grid-area: sidebar">...</aside>
<main style="grid-area: main">...</main>
<footer style="grid-area: footer">...</footer>`}
</CodeBlock>

> **Self-documenting layout** — anyone can read it like a blueprint!


## **Responsive Grids with Media Queries**

Change your grid at different breakpoints.

<CodeBlock
  language="css"
  highlight="8-12"
  collapsible
  defaultCollapsed={false}
>
{`.container {
    display: grid;
    grid-template-columns: 1fr;
    gap: 1rem;
}

/* Tablet & up */
@media (min-width: 768px) {
    .container {
        grid-template-columns: 1fr 1fr;
    }
}`}
</CodeBlock>


## **Auto-Placement: Let the Browser Help**

The browser **automatically places** items if you don’t specify.

Control flow with `grid-auto-flow`:

<CodeBlock
  language="css"
  highlight="2"
  collapsible
  defaultCollapsed={false}
>
{`grid-auto-flow: column;     /* fill columns first */
grid-auto-flow: dense;      /* fill gaps intelligently */`}
</CodeBlock>

Set default size for auto-created rows/columns:

<CodeBlock
  language="css"
  highlight="2"
  collapsible
  defaultCollapsed={false}
>
{`grid-auto-rows: minmax(100px, auto);`}
</CodeBlock>


## **Common Patterns You’ll Use Every Day**

### **Holy Grail Layout**

<CodeBlock
  language="css"
  highlight="3-7"
  collapsible
  defaultCollapsed={false}
>
{`.layout {
    display: grid;
    grid-template-areas:
        "header header header"
        "nav    main   aside"
        "footer footer footer";
    grid-template-rows: auto 1fr auto;
    grid-template-columns: 200px 1fr 200px;
    min-height: 100vh;
}`}
</CodeBlock>

### **Card Grid (Responsive)**

<CodeBlock
  language="css"
  highlight="2"
  collapsible
  defaultCollapsed={false}
>
{`grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));`}
</CodeBlock>

→ Automatically wraps, never too narrow, never too wide.


## **Don’t Do This**

<CodeBlock
  language="html"
  highlight="1-5"
  collapsible
  defaultCollapsed={false}
>
{`<div class="grid">
  <div class="item">...</div>
  <p>Nested text here</p> <!-- Not a grid item! -->
</div>`}
</CodeBlock>

> Only **direct children** are grid items. Wrap content in a child if needed.


## **Quick Testing Checklist**

* Does your layout use `display: grid`?
* Are columns defined with `fr` or `minmax()`?
* Did you test on mobile, tablet, desktop?
* Are named areas clear and semantic?
* Did you use `gap` instead of margins?

Stop fighting with floats.  
Stop nesting flexboxes.  
**Start thinking in grids.**

Your future self (and your teammates) will thank you.
